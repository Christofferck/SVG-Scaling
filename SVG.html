What is a viewbox?
- The viewBox defines the position and dimension, in the user space, of an SVG viewport


What is a viewport?
- The viewport is the part of the webpage that the we can currently see, The scrollbars move the viewport to show other parts of the webpage.


How do they relate to each other?
- They relate to eachother by basicly being being a part of eachother, the viewbox is important in terms of positioning and dimensions, which is a vital part in what the user sees on the page itself


How to scale an SVG drawing's appearance on a web page?
- We make use of Cx which is the X-axis
- We also make use of Cy, which is the Y-axis
- Rx defines the x-axis radius
- and Ry defines the Y-axis radius
- If you can combine all of the four above you should be able to properly scale an SVG on a webpage
